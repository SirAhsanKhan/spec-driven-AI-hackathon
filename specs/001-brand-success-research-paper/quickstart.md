# Quickstart Guide: Brand Success Research Paper

This guide provides the essential steps for a new researcher to get started on this project.

## 1. Tool Setup

### Citation Management

This project uses **Zotero** for citation management.

1.  **Install Zotero**: Download and install the Zotero desktop application from the [official website](https://www.zotero.org/download/).
2.  **Install the Zotero Connector**: Install the Zotero browser extension for your preferred browser. This will allow you to capture sources directly from the web.
3.  **Create a Shared Library**: A shared Zotero library has been created for this project. Please request an invitation to join.

## 2. Research Databases

The following databases should be prioritized for research:

- **[Google Scholar](https://scholar.google.com/)**: For broad discovery of academic papers.
- **[JSTOR](https://www.jstor.org/)**: For access to a wide range of peer-reviewed journals.
- **Business School Case Studies**:
    - [Harvard Business Review (HBR)](https://hbr.org/)
    - [INSEAD](https://www.insead.edu/faculty-research/publications)
    - [Wharton's public repositories](https://knowledge.wharton.upenn.edu/)
- **[ProQuest](https://www.proquest.com/)**: For access to dissertations, theses, and additional academic journals.

## 3. Paper Structure

The paper will follow this high-level structure, as defined in the feature specification:

1.  **Introduction**: State the research question and the paper's thesis.
2.  **Individual Brand Case Studies**: Present the narratives of the selected brand pairs.
3.  **Comparative Analysis**: Compare and contrast the case studies.
4.  **Discussion of Universal Patterns**: Discuss the identified success factors and failure patterns.
5.  **Actionable Insights**: Provide a dedicated section for actionable advice for the business-oriented audience.
6.  **Conclusion**: Summarize the findings and suggest areas for future research.

## 4. Writing and Formatting

- **Format**: All writing should be done in Markdown.
- **Citations**: All citations must be in APA style.
- **Word Count**: The final paper should be between 4000 and 6000 words.
